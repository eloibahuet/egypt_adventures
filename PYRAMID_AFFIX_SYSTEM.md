# 金字塔裝備字綴系統

## 概述
金字塔裝備與普通地圖裝備區隔開來，具有獨特的**字綴系統**和**套裝效果**。

## 套裝系統規則

### 激活條件
1. **三件套**: 必須同時裝備武器、護甲、護符
2. **相同字綴**: 三件裝備必須擁有相同的字綴（如全部為"太陽神拉之"）
3. **相同品質**: 三件裝備必須為相同品質等級
   - ✅ 全精良 (rare)
   - ✅ 全史詩 (epic)
   - ❌ 精良+史詩混搭 (不可)

### 套裝效果顯示
- **裝備面板**: 顯示完整套裝效果和加成詳情
- **狀態面板**: 頂部顯示當前激活的套裝名稱

## 獲取方式

### 金字塔副本掉落
- **普通地圖**: 掉落普通裝備（無字綴）
- **金字塔副本**: 保證掉落帶字綴的裝備
  - 掉落數量: 1-2件
  - 品質分佈: 70% 精良, 30% 史詩
  - 字綴隨機: 從6種字綴中隨機分配

### 字綴分配
- 僅限於**精良**和**史詩**品質
- 普通品質裝備不會獲得字綴
- 字綴完全隨機，可能需要多次挑戰才能湊齊同字綴套裝

```javascript
// 獲取套裝屬性加成

cloneItem(base, rarity, isPyramid = true)
```

### 裝備屬性結構
```javascript
{
  name: "武器名稱",
  rarity: "rare" | "epic",
  isPyramid: true,              // 標記為金字塔裝備
  affix: "ra",                  // 字綴ID
  affixName: "太陽神拉之",      // 字綴顯示名稱
  affixColor: "#FFD700",        // 字綴顏色
  atk: 15,                      // 基礎+品質+字綴加成
  crit_rate: 18,                // 品質額外+字綴加成
  // ... 其他屬性
}
```

## 遊戲策略

### 套裝選擇建議

**攻擊流**: 
- 荷魯斯套裝（高攻擊+連擊）
- 太陽神拉套裝（暴擊流）

**生存流**:
- 阿努比斯套裝（高防禦+閃避）
- 歐西里斯套裝（超高血量+體力）

**平衡流**:
- 伊西斯套裝（幸運+生存）
- 托特套裝（技能+閃避）

### 收集策略
1. 多次挑戰金字塔副本積累裝備
2. 優先收集同一字綴的三件套
3. 史詩套裝強度遠高於精良，但更難湊齊
